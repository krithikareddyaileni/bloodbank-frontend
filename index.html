What is BloodFinder?

BloodFinder is a real-time donor discovery platform designed to connect patients with nearby blood donors instantly. It leverages location-based search and efficient data structures to minimize response time during critical situations.

How It Works

1️⃣ Register as a donor with your location
2️⃣ Search for required blood group
3️⃣ Get nearest donors instantly

Core Features

📍 Location-Based Search

Find donors near your current location using spatial filtering.

⚡ Fast Matching

Uses HashMap and Priority Queue to quickly identify nearest donors.

🧠 Smart Optimization

Grid-based partitioning reduces search complexity and improves efficiency.
